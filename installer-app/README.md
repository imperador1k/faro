# Faro — Desktop Installer UI (`installer-app/`)

> A Vite + React app that powers Faro's branded NSIS installer. Built separately from the main app to keep the installer lightweight (~8MB).

This project is referenced by the Tauri build pipeline and bundled into the final `.exe` installer.

See [src-tauri/installer/README.md](../src-tauri/installer/README.md) for architecture details.
