# Faro — Documentation (`docs/`)

```
docs/
├── architecture/          # System architecture documents
│   ├── knowledge-feed.md  # TikTok-style feed architecture
│   └── ...
```

## Conventions

- Documents are written in English

- Architecture docs describe implemented systems
- Spec docs describe planned or in-progress features
- See [ARCHITECTURE.md](../ARCHITECTURE.md) for the high-level system overview
- See [FARO_MASTER_BLUEPRINT.md](../FARO_MASTER_BLUEPRINT.md) for the comprehensive blueprint
