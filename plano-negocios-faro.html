Sumário Executivo

Oportunidade & Visão

O mercado global de language learning vale +$60 mil milhões e cresce a dois dígitos ao ano. No entanto, 80% dos utilizadores desistem nos primeiros 30 dias. A culpa é do modelo curricular estático: flashcards, repetição espaçada e lições que não dialogam com o mundo real. A Faro nasce para resolver esta crise de retenção.

A Faro é uma plataforma cross-platform (Web + Desktop + Android) de micro-aprendizagem que substitui o currículo fixo por um Knowledge Feed — scroll infinito estilo TikTok de factos reais, notícias e curiosidades, traduzidos e adaptados ao nível CEFR do utilizador em tempo real. O motor é alimentado por Gemini 2.5 Flash + Groq para geração de conteúdos, um pipeline Python/CRON para ingestão de dados globais, e uma arquitetura Next.js + Supabase + Redis que escala horizontalmente desde o primeiro utilizador.

“Tornar a Faro o maior repositório vivo de conteúdos pedagógicos gerados por IA — onde aprender um idioma é um subproduto natural de consumir informação que já te interessa.”

Problema & Solução

O Problema e a Solução

A Crise dos Flashcards

As plataformas atuais (Duolingo, Babbel, Busuu) organizam-se por unidades curriculares lineares. O utilizador completa "Comida" → "Animais" → "Cores" e, quando salta para uma conversa real, não reconhece metade do vocabulário. O modelo é reativo — o conteúdo é decidido meses antes por um curriculum designer, não pelo que está a acontecer no mundo hoje.

Knowledge Feed + Ghost Dictionary

Em vez de lições pré-formatadas, a Faro apresenta um scroll infinito de micro-conteúdos: tweets virais traduzidos, manchetes de news APIs, curiosidades científicas, citações — tudo adaptado ao nível do utilizador. Cada post vem com o Ghost Dictionary: overlay semântico que mostra tradução, contexto gramatical e pronúncia ao toque.

O algoritmo de recomendação (pesos de dificuldade, lacunas de vocabulário, interesses) garante que o conteúdo é sempre desafiante mas não frustrante. O utilizador não "estuda" — consome. E enquanto consome, aprende.

Vantagem Tecnológica

Inovação Tecnológica (A Vantagem Injusta)

A Faro não é uma app de flashcards com IA colada. É uma plataforma de engenharia profunda, desenhada desde o kernel para ser o pipeline de conteúdos mais eficiente do mercado.

Arquitetura cross-platform nativa: Next.js 14 (App Router) no core, Tauri v2 (Rust) no desktop, Capacitor v8 no Android. Um único código-fonte TypeScript compila para Web, Windows, macOS, Linux e Android — 3 plataformas, 1 manutenção.

Pipeline Autónomo de Conteúdos

Um worker Python em CRON ingere dados de 7+ fontes (NewsAPI, Reddit, Wikipedia, Twitter, Giphy, efemérides, cotações) e alimenta uma fila de processamento. O Gemini 2.5 Flash traduz e adapta para 4 níveis CEFR (A1–C2). O Groq faz inferência de baixa latência para interações em tempo real no chat E2EE. O pipeline é fully automated — o admin só revê e aprova.

Stack de Alta Performance

PostgreSQL 15 (35 tabelas normalizadas, RLS com JWTs Clerk), Redis para rate limiting + cache, Drizzle ORM para queries type-safe com zero overhead, Zustand para estado client-side persistido, Server Actions para todas as mutações. Zod em cada action, CSP headers, DOMPurify XSS sanitization, HMAC vault admin.

Porquê o Miguel?

Porque o mercado de language learning está cheio de apps bonitas com backend frágil. A Faro foi construída por alguém que pensa em índices, latência de queries, escalabilidade horizontal e segurança ofensiva antes de pensar em cores ou animações. O resultado é uma plataforma que não precisa de reescrever o backend quando atingir 100 mil utilizadores — porque já foi desenhada para isso desde a primeira linha de código.

Estado atual: 35 tabelas · 34 Server Actions · 131 páginas · 137 componentes · 24 utilitários · 8 hooks · 8 stores · 25+ docs · 4 workflows CI/CD · Docker Compose · Vitest + Playwright. Tudo implementado por uma só pessoa.
